# 接口联调文档

以当前项目代码为准，适用于本地联调。

## 1. 基础说明

- 基础地址：`http://127.0.0.1:8080`
- 统一返回：

```json
{
  "code": 200,
  "msg": "success",
  "data": {}
}
```

- 登录后统一请求头：

```http
token: <jwt>
```

- 不使用 `Authorization: Bearer ...`
- 上传接口当前会根据运行配置返回两种地址：
  - `APP_STORAGE_TYPE=oss`：返回 OSS 公网地址
  - 默认 `local`：返回 `/uploads/...`

## 10. 练功房

### 10.1 获取配置

- `GET /training-room/config`

返回字段：
- `featureOptions`
- `sceneOptions`
- `goalOptions`
- `methodOptions`
- `masterOptions`

### 10.2 上传训练拍摄视频

- `POST /training-room/upload`
- `Content-Type: multipart/form-data`

表单字段：
- `file`
- `folder=training-room`

说明：
- 这是保留的独立上传接口
- 如果前端已经改成在 `/training-room/sessions/{sessionId}/complete` 里直接带 `file`，这里可以不调

返回字段：
- `url`
- `fileName`
- `originalName`
- `size`
- `contentType`

### 10.3 入阵

- `POST /training-room/enter`

```json
{
  "sceneCode": "office",
  "goalCode": "relaxNeck",
  "methodCode": "baduanjin",
  "masterCode": "official",
  "enableScore": true,
  "enablePostureModel": true,
  "enableBreathModel": true
}
```

返回字段：
- `sessionId`
- `planId`
- `title`
- `intro`
- `mantra`
- `sceneLabel`
- `goalLabel`
- `methodLabel`
- `masterLabel`
- `enableScore`
- `enablePostureModel`
- `enableBreathModel`
- `videoId`
- `videoTitle`
- `videoCover`
- `videoUrl`
- `videoDurationSeconds`

### 10.4 提交修习结果

- `POST /training-room/sessions/{sessionId}/complete`

说明：
- 保留原地址不变
- 现在同时支持 `application/json` 和 `multipart/form-data`
- 如果是训练结束后点一次“确认保存”，推荐直接用 `multipart/form-data`
- 这样一次请求就能同时上传拍摄视频并保存评分结果

`application/json` 请求体：
```json
{
  "durationSeconds": 700,
  "caloriesBurned": 120,
  "postureScore": 90,
  "breathScore": 91,
  "postureIssues": ["NECK_TENSE"],
  "breathIssues": ["BREATH_SHALLOW"],
  "practiceVideoUrl": "/uploads/training-room/20260408/demo.mp4",
  "practiceVideoCover": "/uploads/training-room/20260408/demo.jpg",
  "practiceVideoDurationSeconds": 705
}
```

`multipart/form-data` 表单字段：
- `file`：本次训练拍摄视频，可选；传了就会自动先上传再保存记录
- `coverFile`：训练视频封面图，可选
- `durationSeconds`
- `caloriesBurned`
- `postureScore`
- `breathScore`
- `postureIssues`：支持逗号分隔字符串，例如 `NECK_TENSE,CENTER_UNSTABLE`
- `breathIssues`：支持逗号分隔字符串，例如 `BREATH_SHALLOW`
- `practiceVideoUrl`：如果不传 `file`，也可以直接传已上传好的视频地址
- `practiceVideoCover`
- `practiceVideoDurationSeconds`

返回字段：
- `sessionId`
- `title`
- `sceneLabel`
- `goalLabel`
- `methodLabel`
- `masterLabel`
- `totalScore`
- `postureScore`
- `breathScore`
- `scoreLevel`
- `summaryText`
- `adviceItems`
- `recommendedRecipe`
- `recipeReason`
- `practiceVideoUrl`
- `practiceVideoCover`
- `practiceVideoDurationSeconds`
- `durationSeconds`
- `caloriesBurned`
- `completedAt`

## 13. 当前说明

- 接口地址以上述文档为准
- 多数视频类接口都已补齐 `videoUrl` 或 `mediaUrl`
- 视频地址字段名不完全统一：
  - 功法详情：`videoUrl`
  - 功法列表：首页分类列表：`mediaUrl`
  - 社区作品：`mediaUrl`
  - 收藏/点赞/修行日记：`videoUrl`
  - 训练记录/练功房：`practiceVideoUrl`
